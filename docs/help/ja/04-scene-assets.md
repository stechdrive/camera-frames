# シーンアセット

CAMERA_FRAMES のシーンは **スプラット**（Gaussian splatting）と **モデル**（glTF / glb）の 2 種類のアセットで構成されます。各アセットはインスペクターの `Scene` タブで一覧・編集できます。

## 1. シーンマネージャーセクション

インスペクターの `Scene` タブ内、最上部のセクション。

![シーンマネージャー](../assets/screenshots/ja/scene-manager.png)

アセットは **種類ごとに分かれて**表示されます。表示順は固定で `model` → `スプラット` の順。各セクションに種類名と個数のバッジが付きます。

### 1.1 アセットの追加

追加の入口は次の通り。

- **ファイルダイアログ** — ツールレールの `Open Files...`（`Ctrl+O`）
- **ドラッグ & ドロップ** — ビューポートに直接ファイルをドロップ
- **リモート URL** — ツールレールの URL 入力欄に `http://` / `https://` を貼って `Load`
- **パッケージ展開** — `.ssproj` / `.spz` / `.zip` を開くとアセットが展開される

同時に最大 3 つのアセットを並行して読み込みます。読み込みフェーズは `verify → expand → load → apply` の 4 段階で、進捗は進捗オーバーレイに表示されます。

### 1.2 選択する

| 操作 | 効果 |
|---|---|
| クリック | そのアセットを単独選択（再クリックで解除） |
| `Ctrl` / `Meta` + クリック | 加算選択（切替） |
| `Shift` + クリック | 範囲選択（アンカーからの範囲） |

アセット選択は **シーンマネージャー側でもシーン（ビューポート）側でも同期**されます。ビューポート上でスプラットを選択ツールでクリックすると、シーンマネージャー側もハイライトされます。

### 1.3 表示切替（Eye アイコン）

行右端の目アイコン（{{icon:eye}} / {{icon:eye-off}}）で、アセット単位に表示を切替できます。アセットを選択せずに押しても、選択状態には影響しません。

表示オフにしたアセットはビューポート / プレビュー / 書き出しのすべてから消えます。

### 1.4 順序変更

#### ドラッグによる並び替え

行左のグリップアイコンをドラッグして移動します。

- **同じ種類内でのみ並び替え可能**（スプラットをモデルセクションに入れたり、その逆はできない）
- ドロップ位置は行の上半分か下半分かで前後を判定
- 複数選択時は同じ種類のアセット群を一括移動

#### ボタン

各行のコンテキストボタンで 1 段ずつ移動できます（上へ / 下へ）。

### 1.5 ラベル編集

アクティブなアセット行をクリックすると、インライン編集に切り替わります。改行などは自動で正規化されます。

### 1.6 削除

複数選択した状態で `Delete` キー（ビューポートにフォーカスがある時）、またはシーンマネージャーのコンテキスト操作から削除します。削除は元に戻せます。

### 1.7 複製

コンテキスト操作から **Duplicate** を実行。複製アセットは元アセットの直下に挿入されます。ラベルには `Copy`, `Copy 2` のように重複しない接尾辞が付きます。

スプラット編集で加えた変更は複製にも引き継がれます（新しい独立アセットとして分離されます）。

## 2. オブジェクトプロパティセクション

アセット選択時に展開されるプロパティセクション。複数選択時は **一括編集**として働きます。

### 2.1 位置

| フィールド | 刻み |
|---|---|
| X / Y / Z | 0.01 |

複数選択時は変化量が全選択アセットに適用されます。

### 2.2 回転（度）

| フィールド | 範囲 |
|---|---|
| X / Y / Z | −180〜180° |

内部では Euler 角の `XYZ` 順で入出力されます。

### 2.3 スケール

- フィールド: 単一の倍率
- 制約: `0.01 ≤ worldScale`
- 意味: `asset.object.scale = baseScale × worldScale`（読み込み時点のスケールを保ったまま後から倍率を掛ける）

複数選択時は倍率が全アセットに適用されます。

### 2.4 変形を適用

**Apply Transform** ボタンで、現在の外側オブジェクトの変形を中身側へ焼き込みます。焼き込み後、外側オブジェクトは無変形に戻り、`baseScale` は `(1, 1, 1)` になります。

外側と中身の 2 段階構造を 1 段に畳み込むことで、後続の操作が扱いやすくなります。

### 2.5 コンテンツ変形

アセットは 2 段階の構造を持っています。

- **外側オブジェクト**（`asset.object`） — ユーザーが編集する外側の変形
- **中身オブジェクト** — 中身（スプラットの補正グループ、または GLB のルートノード）

コンテンツ変形は中身オブジェクト側の変形であり、通常はユーザーは触らず、`Apply Transform` で焼き込まれる時のみ関わります。

### 2.6 baseScale / worldScale / unitMode

| 項目 | 意味 |
|---|---|
| **baseScale** | 読み込み時点の `asset.object.scale`（固定値） |
| **worldScale** | 後から適用する倍率（ユーザー編集可能） |
| **unitMode** | アセットの想定単位（表示用ラベル） |

単位は種類ごとのデフォルトに基づきます。

### 2.7 書き出しロール

- **beauty** — 通常の書き出し対象（デフォルト）
- **omit** — 書き出し時に別レイヤー / チャネルに分離（非表示扱い）

PSD 書き出し時、`omit` にしたアセットはレイヤーには出力されますが、`beauty` 合成からは外れます。

### 2.8 マスクグループ

任意の文字列で複数アセットを論理的にグループ化する設定。書き出し時に同じグループをまとめて扱えます。空欄なら `—` と表示。

### 2.9 作業ピボット

変形ツールやピボットツールで回転 / スケールの中心点として使う座標。

- **設定** — ピボットツール（`Q`）のギズモをドラッグしてワールド座標を指定
- **リセット** — 原点に戻す。原点と等しいピボットは `null` として扱われる

変形操作をするアセットによって、自然なピボットが異なる（例: 人物モデルなら足元、ビルなら底面中央）ので、ここを明示的に設定すると意図した回転が得られます。

## 3. 照明セクション

シーン全体の照明を 1 つのモデルで制御します。

### 3.1 方向ウィジェット

![照明方向ウィジェット](../assets/screenshots/ja/lighting-widget.png)

132 × 132 px の円形ウィジェット。半径 46 px。中心のドラッグで方向（azimuth / elevation）を指定します。

- **azimuth** — 水平方向の角度（表示中の視点から見た回転、`−180〜180°`）
- **elevation** — 俯仰角（`−89〜89°`）
- 表示中の視点から見た**相対角度**で表現されるため、ビューポートが回転するとウィジェットの向きも相対的に追随する

### 3.2 光源の種類

- **Ambient** — 全体を包む拡散光（強度 `0〜2.5`、初期 `1.1`）
- **Model Light**（主光源） — 方向光
  - **enabled** — ON / OFF（初期 ON）
  - **intensity** — `0〜3`、刻み `0.01`、初期 `2.0`
  - **azimuth / elevation** — 上記ウィジェットで設定
- **Ambient fill** — 主光源の反対方向からの補助光（自動）

### 3.3 リセット

`Reset Lighting` で既定値（ambient 1.1、intensity 2.0、azimuth 36.87°、elevation 45°）に戻せます。

## 4. スプラットとモデルの違い

| | スプラット | モデル |
|---|---|---|
| **主なフォーマット** | `.ply` / `.spz` / `.splat` / `.kSplat` / `.sog` / `.rad` / `.zip` | `.glb` / `.gltf` |
| **構造** | SplatMesh（補正グループ付き）| GLB シーン / ノード |
| **スプラット編集** | 対応 | 非対応 |
| **書き出し** | 通常 / スプラットレイヤー（PSD のみ） | 通常 / モデルレイヤー（PSD のみ） |
| **中身オブジェクト** | 補正グループ | GLB のルートノード |

スプラット編集（`Shift+E`）はスプラットアセットに対してのみ有効で、モデルには適用されません（別アセットとして扱うか、3D ツール側で編集してから再読み込みしてください）。

## 5. 削除・複製とスプラット編集

- **削除** — 選択アセットを削除。元に戻せる。関連するスプラット編集の選択状態はクリアされる
- **複製** — 元アセットの直下に新アセットを複製。スプラット編集の選択状態は引き継がれない

スプラット編集の `Separate` / `Duplicate` から作られた新アセットは、元アセットの**直上**に挿入されます（[スプラット編集](09-per-splat-edit.md) で詳述）。

## 6. 関連ショートカット

シーンアセット固有のショートカットは少なく、主にビューポート側の操作と連動します。

- `Delete` / `Backspace` — 選択アセットを削除
- 選択時の修飾キー（`Shift` / `Ctrl` / `Meta`）

全ショートカットは [キーボードショートカット一覧](11-shortcuts.md) を参照。
