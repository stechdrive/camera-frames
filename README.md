# CAMERA_FRAMES

CAMERA_FRAMES app built on Spark 2.0 preview.

## Repo Role

This repository is the app itself.
Spark is consumed as an external dependency from the official `v2.0.0-preview` line.

## Development

```powershell
npm install
npm run develop
```

## Validation

```powershell
npm run lint
npm run build
npm test
```
